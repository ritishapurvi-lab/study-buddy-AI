# 📚 Study Buddy AI

Study Buddy AI is a Python project that helps students create a personalized study schedule based on the subject, number of chapters, and days remaining before an exam.

## Features

- Takes subject name as input
- Takes number of chapters
- Takes number of days left
- Calculates chapters to study each day
- Generates a day-wise study plan

## Concepts Used

- Variables
- User Input
- Arithmetic Operators
- for Loops
- range()
- Variables and Calculations

## Future Improvements

- Save study plans to a file
- Add revision days
- GUI using Tkinter
- AI-powered study recommendations
- Integration with Gemini/OpenAI APIs
